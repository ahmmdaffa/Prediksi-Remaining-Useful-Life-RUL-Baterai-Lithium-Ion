# Prediksi Remaining Useful Life (RUL) Baterai Lithium-Ion Berbasis KNN

Proyek *data mining* ini dirancang untuk memprediksi sisa masa pakai (*Remaining Useful Life* - RUL) baterai Lithium-Ion pada kendaraan listrik. Dikembangkan sebagai solusi teknologi untuk mengatasi kekhawatiran konsumen (*range anxiety*) serta mendukung akselerasi ekosistem kendaraan listrik (EV) di Indonesia.

Proyek ini dibangun secara *end-to-end*, mencakup analisis data, pelatihan model machine learning, hingga implementasi berupa *Custom Web App* yang memisahkan *Backend API* dan *Frontend UI*.

---

## 🚀 Fitur Utama
- **Eksperimen Data Komprehensif:** Tahapan EDA, *preprocessing*, *feature scaling*, hingga evaluasi model terdokumentasi di Jupyter Notebook.
- **Prediksi Akurat:** Menggunakan algoritma **K-Nearest Neighbors (KNN) Regressor** yang dioptimalkan untuk data non-linear.
- **Backend API Modern:** Menggunakan **FastAPI** yang ringan, cepat, dan otomatis menyediakan dokumentasi interaktif.
- **Frontend Responsif:** Antarmuka web minimalis berbasis HTML/CSS murni untuk interaksi pengguna yang mulus tanpa *reload* halaman.
